# 基于 Tensorflow.js 的实践 Demo

引入Tensorflow的PoseNet姿势预测模型, 进行人体关键点位的检测, 然后对输入对象的图像进行拷贝到姿势预测的容器内, 然后在姿势预测内绘制关键点位的骨架, 从而进行简单的姿势预测.(我这里因为没有摄像头因此放弃了真实的视频源作为输入对象, 只用简单的视频作为模拟, 可参考原文作WebRTC流传递)

有了个这个Demo实践, 其实可以拓展很多, 比如运动健身的时候姿势的正确性, 可以做保存做虚拟人物, 也可以做Switch上大火的健身环又系, 甚至VR, AR等.
